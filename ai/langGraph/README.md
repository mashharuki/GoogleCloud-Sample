# LangGraphを検証するフォルダ

## セットアップ

1. travy.com でAPIキーを作成。

2. Open AIかOpenRouterのAPIキーを作成する。

2. 環境変数をセットアップする。

    ```txt
    OPENAI_API_KEY="sk-..."
    TAVILY_API_KEY="tvly-..."
    ```
