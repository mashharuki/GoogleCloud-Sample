# LangChain × Gemini × Vertex AI

## 動かし方

- セットアップ

    `.env`を作成する。

    ```txt
    GEMINI_API_KEY=""
    ```

- インストール

    ```bash
    bun install
    ```

- サンプルを試す

    ```bash
    bun run dev
    ```
